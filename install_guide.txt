꽃님이의 갤러그 게임 설치 가이드
================================

1. 시스템 요구사항
------------------
- 운영체제: Windows 10 이상
- 메모리: 최소 2GB RAM
- DirectX: 9.0 이상
- 저장공간: 최소 100MB의 여유 디스크 공간

2. 설치 방법
------------
1. 다운로드한 main.exe 파일을 더블클릭하여 실행합니다.
2. Windows 보안 경고가 표시될 경우:
   - '추가 정보' 클릭
   - '실행' 버튼 클릭
   - '알 수 없는 앱' 실행 허용 선택

3. 조작 방법
------------
- 방향키: 우주선 이동
- 스페이스바: 발사
- ESC: 게임 종료

4. 게임 규칙
------------
- 적을 파괴하여 점수를 획득하세요
- 3개의 목숨이 주어집니다
- 스테이지가 진행될수록 난이도가 증가합니다

5. 문제 해결
------------
게임 실행에 문제가 있는 경우:
1. DirectX를 최신 버전으로 업데이트하세요
2. 그래픽 드라이버를 최신 버전으로 업데이트하세요
3. 관리자 권한으로 실행해보세요
4. Windows 보안 설정에서 앱 실행 허용 확인

6. 제작자 정보
------------
- 제작자: AI 교육 강사 성근쌤
- 문의: support@example.com
- 웹사이트: www.example.com

7. 주의사항
------------
- 게임 실행 시 Windows 보안 경고가 표시될 수 있습니다.
- 이는 게임의 무결성을 확인하는 정상적인 과정입니다.
- 게임 파일을 수정하거나 삭제하지 마세요.
- 안티바이러스 프로그램이 게임 실행을 차단할 수 있습니다.
- 이 경우 예외 설정에 게임을 추가해주세요.
